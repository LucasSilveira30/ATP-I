Esse é um trabalho sobre programação de menus interativos em aplicações de linha de comando.

O trabalho é composto por dois exercícios: 
• Exercício 1: Monte o seu poke! → Crie um código funcional que mostre para o usuário um menu interativo de um restaurante de
  comida Havaiana.
• Exercício 2: Passagem de ônibus. → Crie um programa que mostre para o usuário um menu interativo de uma empresa de transporte
  rodoviário.
